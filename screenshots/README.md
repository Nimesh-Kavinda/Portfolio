# Screenshots Directory

This directory is meant to contain screenshots of your portfolio website for the README.md file.

## Required Screenshots:

1. **homepage-light.png** - Homepage in light mode
2. **homepage-dark.png** - Homepage in dark mode  
3. **about-section.png** - About section with GitHub integration
4. **services-section.png** - Services section
5. **github-repos.png** - GitHub repositories showcase
6. **mobile-responsive.png** - Mobile responsive design
7. **contact-section.png** - Contact form section

## How to Take Screenshots:

1. Open your portfolio website in a browser
2. Use browser developer tools to test different screen sizes
3. Take screenshots of each section as listed above
4. Save them with the exact filenames listed above
5. Ensure images are high quality (PNG format recommended)

## Image Guidelines:

- **Resolution**: Minimum 1200px width for desktop screenshots
- **Format**: PNG preferred for sharp text and UI elements
- **Mobile screenshots**: Use 375px width (iPhone standard)
- **File size**: Keep under 1MB each for faster loading

Once you add the actual screenshots, you can delete this instruction file.
